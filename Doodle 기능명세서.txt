[Doodle] 통합 기능 명세서 (Final Full Version)

1. 메인 지도 (Map Interaction)
F-1.1 기본 지도: 카카오맵 API 연동 및 사용자 현재 위치(Geolocation) 표시.
F-1.2 퀵 필터 (비로그인 가능): 지도 화면 내 즉시 노출되는 토글 버튼으로 '흡연부스/공중화장실/스팟' 마커를 개별/전체 온오프.
F-1.3 클러스터링: 지도 레벨 축소 시 마커 그룹화 표시 및 클릭 시 해당 영역 확대.
F-1.4 중복 좌표 스태킹 (Stacking):
동일 위/경도 좌표에 데이터가 여러 개일 경우, 최초 등록 마커 노출.
마커 우측 상단에 +[중복 개수] 배지(Badge) 표시 (예: +5).
마커 클릭 시 해당 좌표의 모든 데이터를 보여주는 무한 스크롤 리스트 팝업 노출.

2. 검색 및 해시태그 (Search & Hashtag)
F-2.1 로컬 자동완성: 앱 진입 시 전체 태그 리스트를 fetch하여 클라이언트 로컬 메모리에 저장. 입력 시 즉각적인 초성/부분 일치 검색 제공 (서버 부하 방지).
F-2.2 실시간 순위 오버레이:
검색창 클릭 시 지도를 가리는 전용 오버레이 레이어 활성화.
서버에서 집계된 실시간 인기 해시태그 TOP 10 리스트 노출.
리스트는 하단으로 스크롤 시 무한 스크롤로 추가 순위(최대 30위 등) 확장 가능.
태그 클릭 시 오버레이가 닫히며 해당 검색 결과가 반영된 지도로 이동.

3. 게시글 및 콘텐츠 (Posts & Content)
F-3.1 흡연부스/스팟 등록: 지도 위 지점 클릭 시 위/경도 자동 매핑. 이름, 설명, 해시태그 입력 기능.
F-3.2 공중화장실 연동: 공공데이터 API를 통해 위치 마커 자동 생성. 클릭 시 주소 및 운영 시간 상세 정보 제공.
F-3.3 상세 페이지/팝업: 마커 클릭 시 나타나는 정보창 내에서 무한 스크롤을 통한 데이터 탐색 지원.

4. 유저 및 커뮤니티 (User & Community)
F-4.1 회원 인증: 로그인 및 회원가입(아이디, 비번, 닉네임).
F-4.2 마이페이지:
내가 등록한 흡연부스/스팟 기록 리스트 (무한 스크롤 적용).
내가 작성한 글 및 사용한 해시태그 모아보기.
F-4.3 신고 시스템: 마커 상세 UI 내 신고 버튼. 사유 선택(정보 불일치, 장소 사라짐 등) 후 관리자 전송.

5. 관리자 시스템 (Admin)
F-5.1 통합 대시보드: 실시간 등록 건수 현황판.
F-5.2 신고 알리미: 유저 신고 내역 실시간 리스팅 및 특정 마커 게재 중단 처리.
F-5.3 해시태그 모니터링: 부적절하거나 어뷰징이 의심되는 태그를 실시간 순위 및 자동완성에서 즉시 게재 중단 처리.
