[include 전환 메모]

이번 수정은 iframe 전체 제거가 아니라, 메인 조각 UI를 include 방식으로 바꾼 버전입니다.

바꾼 대상
- search.html  -> partials/search.partial.html + App.pageSearch
- ui.html      -> partials/ui.partial.html + App.uiCategorySelector
- ranking.html -> partials/ranking.partial.html

유지한 대상
- mypage.html iframe 유지
- register.html iframe 유지

핵심 파일
- main.html
- js/core/include.loader.js
- partials/*.partial.html
- js/pages/page.search.js
- js/ui/ui.category.selector.js
- js/pages/page.main.js

포인트
1) data-include 속성으로 파셜 HTML을 불러옴
2) data-init 속성으로 파셜 삽입 뒤 init 함수 실행
3) search / category 는 iframe 모드 + include 모드를 둘 다 지원하도록 수정
4) CSS 충돌을 줄이기 위해 ui.css 에 .ui-inline-root 범위를 추가

주의
- file:// 로 열면 브라우저 보안정책 때문에 fetch(include)가 막힐 수 있음
- 가능하면 Live Server 같은 로컬 서버로 실행하는 것이 안전함
