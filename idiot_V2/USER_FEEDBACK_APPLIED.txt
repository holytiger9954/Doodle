이번 수정에서 반영한 핵심

1) 기본 지도 마커 정책 변경
- 카테고리를 누르기 전에는 기본 카테고리 마커 + 등록된 사용자 마커를 모두 보여주도록 변경
- 검색 초기화 시에도 다시 전체 마커 상태로 복귀

2) 마이페이지를 사용자 전용 공간으로 정리
- 기존 공용 카테고리 목록 제거
- 마이페이지에는 '찜 목록'과 '내가 등록한 장소'만 표시
- 찜 목록은 로그인 사용자별 localStorage 분리 저장
- 정보 팝업에서 찜하기 / 찜해제 가능

3) 검색 UX 정리
- 검색창 바로 아래 인라인 경고 문구 제거
- 입력창 포커스 시 결과 오버레이를 먼저 열고 안내 문구 노출
- 검색어 입력/검색 실행 이후 결과 목록으로 자연스럽게 전환
