마이페이지 로그인 게이트 수정 메모

- 문제: 로그인 관련 UI(loginwindow)는 파일만 남아 있고, 실제 마이페이지와 연결되지 않아 비로그인 상태에서도 마이페이지 기능이 바로 노출됨.
- 해결: mypage.html 상단에 로그인 상태 패널을 직접 추가하고 page.mypage.js로 로그인 전/후 상태를 제어함.
- 로그인 전:
  - "로그인이 필요합니다" 문구 노출
  - 로그인 / 회원가입 버튼 노출
  - 카테고리 버튼 클릭 시 목록 대신 안내 문구 강조
- 로그인 후:
  - 닉네임 환영 문구 노출
  - 로그아웃 / 비밀번호 변경 버튼 노출
  - 카테고리 버튼 정상 동작
